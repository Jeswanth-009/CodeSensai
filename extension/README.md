# CodeSensei IDE

AI Reasoning Coach for Indian Developers — Powered by AWS Bedrock.

This VS Code extension explains code in four layers (what, why, pitfalls, Socratic) with support for Hinglish analogies, complexity analysis, and multi‑turn tutoring. It works with any file with functions and displays results in a sidebar.

## Installation
Install from the Visual Studio Marketplace:

https://marketplace.visualstudio.com/items?itemName=JeswanthSaiKancharana.codesensei-ide

or run the following from the command line:

```bash
code --install-extension JeswanthSaiKancharana.codesensei-ide
```

## Usage
1. Select a code snippet or place the cursor inside a function.
2. Right-click and choose **Ask CodeSensei** or click the CodeLens button above the function.
3. Read the structured explanation in the sidebar and optionally export it to Markdown.

For full documentation, refer to the [project README](../README.md).
